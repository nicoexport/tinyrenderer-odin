# tinyrenderer-odin

This is a small private project implementing the tiny renderer as described in this series of articles: [Software rendering in 500 lines of bare C++](https://github.com/ssloy/tinyrenderer).

The purpose of this project is for me to lear the fundamentals of the Odin programming language.

 I did a similar thing for learning the basics of Zig which can be found [here](https://github.com/nicoexport/tinyrenderer-zig-raylib). I didnt finish the tiny renderer in Zig but plan to do so in Odin.
